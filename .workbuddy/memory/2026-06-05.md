## Tailwind CSS 基础
- 什么是 Tailwind：用简短工具类名代替 CSS 属性
- 命名规律：属性前缀 + 数值（间距×4）+ 方向后缀
- 常用类名：flex, items-center, gap, text-, bg-, rounded
- 响应式前缀：sm:, md:, lg:, xl:
- 文档查询：tailwindcss.com
