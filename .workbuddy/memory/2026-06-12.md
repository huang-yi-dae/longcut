## 追加：2026-06-12 第二次学习 — React Hooks 进阶

### 今日知识点
- **useEffect**：`[]` 只跑一次，`[a,b]` 依赖变重跑，`return fn` 清理函数（清定时器/移监听/断连接）
- **useRef**：跨渲染保留值，变了不触发重渲染（vs useState 变了触发渲染）
- **addEventListener / removeEventListener**：标准配对模式，必须是同一个函数引用
- **useCallback**：缓存函数跨渲染保持引用，等价于 `useMemo(() => fn, deps)`
- **useMemo**：缓存计算结果，依赖不变直接复用
- **自定义 Hook**：把 hook 逻辑封装成 `useXxx` 函数（useElapsedTimer、useInAppBrowser、useScrolled）
- **数据获取标准模式**：useState 存 + useEffect 取

### 用户关键提问
- "useCallback是函数版本的useRef，使用的是useEffect的语法。"
- "useState用来控制和改变一个状态，useEffect决定具体什么时候来控制和改变这个状态。"
