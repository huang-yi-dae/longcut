- 改造 `components/auth-modal.tsx`：`createClient()` 改为 `useMemo(() => hasSupabaseConfig() ? createClient() : null, [])`
- 改造 `app/settings/settings-form.tsx`：同上模式保护

## Stage 5.1 — React Hooks 深入

- **`useEffect`** = 条件执行器（控制代码何时执行，不是缓存）
- **`useMemo`** = 值缓存（依赖不变复用上次结果）
- **`useCallback`** = 函数缓存（依赖不变返回同一引用，让 `===` 通过）
- 关键理解：React 用 `===` 比较函数引用，每次渲染 `() => {}` 都是新对象 → 子组件白重渲染
- 用户易混淆"执行代码"和"渲染"，需明确区分
- 用户提醒：不要一次性给太多内容
